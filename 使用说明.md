# Wenni私厨使用说明

## 已支持功能

- 微信 H5 首页：Wenni私厨、今日现做、开始点餐。
- 菜品卡片：图片、菜名、星级、制作时长、食材、加入订单。
- 分类导航：全部、招牌菜、早餐、晚餐、快手菜、预约菜。
- 今日推荐：自动从招牌、孩子爱吃、快手等菜品里推荐。
- 家庭成员：爸爸、妈妈、女儿、儿子。
- 订单状态：已下单、制作中、已完成、已领取。
- 私厨后台：新增、编辑、删除菜品，查看订单，更新状态。
- 本月统计：点单次数、最受欢迎菜品、最爱点单成员等。
- 智能推荐：根据现有食材匹配可做菜品。
- 图片上传：可上传菜品图片到 Supabase Storage。
- 今日已点：家人端可以看到今天所有人已经点过的菜单。
- 偏好分析：后台可按本周、本月、今年统计热门菜品和家人喜好。
- 默认菜品图：默认菜库已配好线上食物图片，没图的云端菜品会自动补图。

## 更新 Supabase

如果你之前已经执行过旧版 SQL，请再执行一次新版 `supabase-schema.sql`。

这不会清空原来的菜单和订单，只会补齐：

- 新订单状态默认值
- dish-images 图片桶
- 图片上传权限

## 更新 GitHub Pages

需要重新上传或替换这 3 个文件：

- `index.html`
- `config.js`
- `supabase-schema.sql`

其中真正影响网页的是：

- `index.html`
- `config.js`

`supabase-schema.sql` 是给你以后维护数据库用的。

如果只是更新这次页面体验，至少替换：

- `index.html`

打开新版网页后，系统会自动把默认菜品图片补到云端菜库里。

## 上传步骤

1. 打开 GitHub 仓库。
2. 点 `Add file`。
3. 选择 `Upload files`。
4. 拖入新版 `index.html` 和 `config.js`。
5. 如果页面提示同名文件已存在，选择替换。
6. 点 `Commit changes`。
7. 等 1-3 分钟后刷新 GitHub Pages 网址。
